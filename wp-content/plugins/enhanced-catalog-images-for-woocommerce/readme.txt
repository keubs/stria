Enhance your WooCommerce catalog images with multiple effects.

== Description ==

View the [Enhanced Catalog Images for WooCommerce demo](https://demos.iconicwp.com/enhanced-catalog-images-for-woocommerce/fade/).

Enhanced Catalog Images for WooCommerce adds a selection of popular effects for your shop's catalog images.

### Key Features

The free plugin comes with some fantastic features for your catalog images:

* **Fade on Hover**  
  Fade to the product's secondary image on hover.
* **Horizontal Slide**  
  Slide horizontally through the product's images.
* **Vertical Slide**  
  Slide vertically through the product's images.

### Upgrade to Premium

With the premium version of Enhanced Catalog Images for WooCommerce, you will get the following additional effects:

* **Bullets**  
  Use bullets to cycle through the product's images.
* **Thumbnails**  
  The product's additional images will be displayed as thumbnails. Clicking on a thumbnail switches the main image.
* **Zoom**  
  The product image will zoom on hover.
* **Picture in Picture**  
  Switch between the primary and secondary image via a Picture in Picture effect.
* **Modal Gallery**  
  Open all the product's images in a modal gallery.
* **Enlarge**  
  Enlarge the product's image on hover.
* **Flip**  
  Flip effect on hover to the product's secondary image.

Simply install the free version to try it out, and then click **Upgrade** from the settings page.

### Options

Most of the effects have options that can be configured to achieve a look unique to your WooCommerce store. Change the icon colours, images to show, transition timings and effects, and more.

== FAQ ==

### My images are no different, or show up twice

Some themes hard code their images into the WooCommerce template files. If this is happening for you, follow [these instructions](https://iconicwp.ticksy.com/article/8591/).

== Installation ==

To install the plugin:

### Via WordPress Admin

1.  Navigate to Plugins > Add New.
2.  Search for \"Enhanced Catalog Images for WooCommerce\".
3.  Find the plugin by \"Iconic\".
4.  Install and activate the plugin.

### Via Download

1.  Navigate to Plugins > Add New > Upload.
2.  Click Choose File, and choose the file enhanced-catalog-images-for-woocommerce.zip.
3.  Once uploaded, click activate plugin.
4.  The plugin is now installed and activated.
